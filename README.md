# Dir Media Viewer

ディレクトリ内の画像・動画をグリッドで一覧表示することのみにこだわった軽量メディアビューワー。
ファイル操作はOSのファイルシステムで行うことを前提として、ディレクトリ内のメディアの閲覧に特化している。

## セットアップ

```sh
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

## 使い方

### 基本操作

1. ブラウザにメディアを含むディレクトリをドラッグ&ドロップ
2. グリッドに画像・動画が一覧表示される
3. 動画はグリッド上で無音ループ再生
4. アイテムをクリックで拡大表示

### ディレクトリ移動モード

ドラッグ&ドロップでディレクトリを読み込むと、左側にディレクトリツリーのドロワーが自動で開く。
ドロップしたディレクトリをルートとしたツリーを操作して、サブディレクトリを自由に閲覧できる。

#### ツリー操作キー（ドロワー表示中）

| キー | 機能 |
|------|------|
| `↑` `↓` | ツリーカーソルを上下移動 |
| `←` / `Shift`単体 | 親ディレクトリへカーソル移動 |
| `→` | 子ディレクトリへカーソル移動（展開して最初の子に移動） |
| `Space` | カーソル位置のディレクトリを開く（グリッドにファイル表示） |
| `Shift` × 2 | ドロワーを閉じる（閲覧モードに戻る） |

#### モード切替

- **ドロワー表示中**: ツリー操作キーが有効。通常のキーバインド（R, A, I, V, 1-3）も併用可能
- **ドロワー非表示時**: 通常の閲覧モード。キーバインドはすべて従来通り。`Shift` × 2 でドロワーを再表示
- ドロワーを閉じても、選択中のディレクトリのファイルはそのまま表示される

### ツールバー機能

| 機能 | 説明 |
|------|------|
| Recursive | 2階層下のサブディレクトリまで再帰的にメディアを読み込む |
| All / Image / Video | メディアタイプでフィルタリング |
| 大 / 中 / 小 | グリッドサイズ切替 |
| Clear | 表示をクリアして初期画面に戻る |

### キーボードショートカット（閲覧モード）

| キー | 機能 |
|------|------|
| `R` | Recursive トグル |
| `A` | フィルター: All |
| `I` | フィルター: Image |
| `V` | フィルター: Video |
| `1` | グリッド: 大 |
| `2` | グリッド: 中 |
| `3` | グリッド: 小 |
| `Esc` | 表示クリア |

### 拡大表示

| キー | 機能 |
|------|------|
| `←` / `h` | 前の画像 |
| `→` / `l` | 次の画像 |
| `Space` / `Esc` | 拡大表示を閉じる |
| `M` | 原寸 / フィット切替 |
| `D` | ダウンロード |

- ダウンロードボタンで再エンコード（メタデータ除去）してダウンロード
  - 画像 → JPEG
  - 動画 → MP4

### 設定の保持

以下の設定はlocalStorageに保存され、次回アクセス時に復元される:

- グリッドサイズ（大/中/小）
- ダイアログ表示サイズ（原寸/フィット）
- メディアフィルター（All/Image/Video）

Recursiveモードは毎回OFFで開始する。

## Electron デスクトップアプリ

### 開発

```sh
# ブラウザで開発（HMR付き）
npm run dev

# Electronで確認（ビルド済みSPAを読み込み）
npm run electron:dev

# 開発サーバー接続モード（HMR付き、別ターミナルで npm run dev を先に実行）
npm run electron:preview
```

### ビルド（配布用パッケージ生成）

```sh
npm run electron:build
```

`release/` ディレクトリにmacOS用 `.dmg` が生成される。

## 前提条件

- Node.js 18+
- 再エンコード機能を使う場合: ffmpeg がシステムにインストール済みであること

## 技術スタック

- SvelteKit (Svelte 5)
- TailwindCSS v4
- Bits UI v2
- Lucide Svelte
